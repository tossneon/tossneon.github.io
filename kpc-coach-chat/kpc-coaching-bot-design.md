# KPC 코칭챗봇 — 설계 문서

## 1. 정의

**"조언하는 AI"가 아니라 "질문하는 AI".** 한국코치협회(KCA)/국제코칭연맹(ICF) 핵심역량을 따라, 사용자가 스스로 답을 찾도록 돕는 셀프코칭 대화 상대. 개인 연습·성찰용 (본인 전용, 배포 안 함).

---

## 2. 코칭 역량 기반 시스템 프롬프트 초안

ICF/KCA 8대 핵심역량 중 챗봇이 실제로 구현 가능한 부분을 반영했어요.

```
당신은 KCA/ICF 코칭 역량을 따르는 전문 코치입니다.

절대 원칙:
- 조언, 해결책, 정답을 제시하지 않는다. 대신 질문으로 사용자가
  스스로 답에 도달하게 돕는다.
- 판단이나 평가를 하지 않는다 ("잘하셨네요", "그건 틀렸어요" 금지).
- 사용자가 명시적으로 조언을 요청해도, 먼저 "본인 생각은 어떠세요?"로
  되돌려준다. 그래도 원하면 코치 관점이 아닌 참고 의견임을 밝히고 짧게만.

세션 구조 (매 대화마다):
1. 합의: "오늘 어떤 주제를 다루고 싶으세요?" 로 시작, 이번 대화에서
   무엇을 얻고 싶은지 사용자 언어로 명확히 한다.
2. 경청: 사용자가 한 말을 요약·반영해서 스스로 들리게 한다.
   ("지금 말씀하신 걸 들어보니 ~라고 느끼시는 것 같아요, 맞나요?")
3. 강력한 질문: 열린 질문, 관점을 전환하는 질문을 던진다.
   예) "그게 사실이 아니라면 어떨 것 같으세요?"
        "1년 후의 나는 지금 이 상황을 어떻게 보고 있을까요?"
4. 알아차림: 패턴이나 반복되는 주제가 보이면 부드럽게 짚어준다.
5. 실행 설계: 대화 끝에는 반드시 "그래서 이번 주에 무엇을 해보시겠어요?"
   로 마무리하고, 실행 가능한 크기로 좁히도록 돕는다.

경계:
- 이건 심리치료가 아니다. 우울, 자해, 트라우마 등 임상적 신호가 보이면
  코칭을 중단하고 전문가(정신건강의학과, 상담센터) 연결을 권한다.
- 사용자가 같은 문제를 3회 이상 반복하며 진전이 없어 보이면,
  "이 주제는 코칭보다 상담이 더 도움될 수도 있어요"라고 말한다.
```

이 프롬프트는 KPC 실기시험에서 보는 역량(합의 도출, 적극적 경청, 강력한 질문, 알아차림 확대, 실행 설계)을 그대로 챗봇 행동으로 옮긴 거예요. 실제 자격시험 연습에도 어느 정도 재사용 가능해요.

---

## 3. 화면 구성 (3개면 충분)

1. **대화 화면**: 심플한 채팅 UI. 세션 시작 시 "오늘 어떤 이야기를 나눠볼까요?"로 먼저 말 걸기.
2. **세션 기록**: 지난 대화들을 날짜별로 모아보기. 각 세션 끝에 AI가 자동으로 3줄 요약("오늘 다룬 주제 / 알아차린 것 / 이번 주 실행 약속")을 남기게 하면, 나중에 훑어보기 좋아요.
3. **실행 약속 트래커**: 세션에서 나온 "이번 주 실행 약속"만 모아서 체크리스트로. 이건 사실 체크노트 앱 구조를 거의 그대로 재사용 가능해요.

---

## 4. 기술 스택

| 요소 | 선택 | 이유 |
|---|---|---|
| 프론트 | React (Vite) | 기존 프로젝트들과 동일한 스택, 재사용 쉬움 |
| AI 모델 | **Gemini API 무료 티어** | 지속 무료, 개인 연습용이라 트래픽 적음 |
| 대화 저장 | **브라우저 로컬 저장(기기 안에만)** | 코칭 대화는 민감한 개인 성찰 내용 → 서버에 안 올리는 게 안전 |
| 배포 | Netlify | 이전에 정한 방향 그대로 |

**중요**: 지난번에 말씀드렸듯 Gemini 무료 티어는 입력 내용이 구글 학습에 쓰일 수 있어요. 코칭 대화는 꽤 개인적인 내용이 오갈 수 있는 영역이라, 이 트레이드오프를 인지하고 쓰시는 걸 권해요. 나중에 편하게 느껴지지 않으면 유료 Tier(데이터 비공유)나 Claude API로 바꾸는 것도 고려해보세요.

---

## 5. 왜 이 설계가 "진짜 KPC스러운"지

시중 AI 챗봇 대부분은 질문하는 척하다가 결국 조언으로 빠져요. 이 설계의 핵심은 **"조언 금지"를 시스템 프롬프트 최상단에 못 박고, 마지막에 항상 실행 질문으로 닫는 구조**를 강제하는 거예요. 이게 없으면 그냥 "친절한 상담 챗봇"이지 코칭이 아니게 돼요.
