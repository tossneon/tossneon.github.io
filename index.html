Claude 자동 허용

Claude Desktop에서 권한 팝업이 뜨면 "항상 허용"을, 없으면 "한 번만 허용"을 자동으로 눌러주는 매크로. 좌표가 아니라 버튼 글자를 읽어서 누르므로 창 위치·해상도가 바뀌어도 동작합니다.

⬇ 다운로드 (.zip)Windows 10/11 · 별도 설치 불필요

받은 뒤 압축 풀고 control-panel.vbs 더블클릭이면 끝

⚠️ 먼저 읽어주세요

이 매크로는 MCP 도구 실행 확인을 사람 대신 자동 승인합니다. 신뢰할 수 없는 MCP 서버가 붙어 있으면 위험한 동작(파일 삭제·외부 전송 등)도 그냥 통과되니, 믿을 수 있는 환경에서만 켜세요.

이렇게 씁니다

  1. 1위 버튼으로 claude-auto-allow.zip 다운로드
  2. 2압축 풀기 전 우클릭 → 속성 → "차단 해제" 체크 (인터넷 파일 경고 방지)
  3. 3control-panel.vbs 더블클릭 → 조작 창이 뜹니다
  4. 4▶시작 / ■정지로 켜고 끄기. 자동 클릭 횟수와 기록이 실시간으로 보입니다

특징

  • 버튼 글자로 찾아 클릭 — 창 위치/해상도 바뀌어도 OK
  • 현황 표시 + 시작/정지 버튼이 있는 조작 화면
  • Windows 내장 PowerShell만 사용 — AutoHotkey 등 별도 설치 불필요
  • 조작 창을 닫으면 완전히 종료 (백그라운드 잔여 없음)
